Vivek and Prime Numbers

Author: savaliya_vivek
Time Limit: 1 second
Source Limit: 50000 Bytes
Input: Standard Input
Output: Standard Output
Languages: C,CPP14,JAVA,PYTH,PYTH 3.6
Question

Vivek likes prime numbers but hi so weak in maths,so he not able to decide whether given number is prime or not. So, He take help of his friend Mayur for check given number n is prime or not.

Mayur is good Mathematician,so he is able to decide given number is prime or not.

Input

The first line contains an integer T,total number of testcases.Then follow T lines,each line contains an integer N.

Output

print ans in single line if given number is prime then print "Prime",otherwise "Not Prime".

Constraints

1≤ T ≤1000
1≤ N ≤1015
Example

Input
2
3
4
Output
Prime
Not Prime
