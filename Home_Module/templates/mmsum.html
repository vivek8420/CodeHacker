Mayur and Max Sum

Author: mayur_1439
Time Limit: 1 second
Source Limit: 50000 Bytes
Input: Standard Input
Output: Standard Output
Languages: C,CPP14,JAVA,PYTH,PYTH 3.6
Question

Vivek want to check programming skill of mayur. So, He gives sequence A1,A2,........,An of n elements to Mayur and give task to choose some elements in sequence and maximize sum of chosen elements, but mayur can not choose adjecent element.

If two element are called adjecent, if they share side.

Input

The first line contains an integer T,total number of testcases.

For each test cases, first line N ,no of elements.
Second line N space separated numbers.

Output

Print One integer for each test cases , maximum some of chosen elements.

Constraints

1 ≤ T ≤ 1000
10 ≤ N ≤ 105
10-5 ≤ Ai ≤ 105 where 1 ≤ i ≤ N
Example

Input
1
4
1 2 3 4
Output
6
Explanation:

here, only total possiblity of chosen elements are 1, 2 ,3 ,4 , (1,3), (1,4), (2,4). But Maximum sum is set (2,4),so answer is 2 + 4 = 6.
