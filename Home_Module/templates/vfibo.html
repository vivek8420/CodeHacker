Vivek and Fibonaki

Author: savaliya_vivek
Time Limit: 1.5 second
Source Limit: 50000 Bytes
Input: Standard Input
Output: Standard Output
Languages: C,CPP14,JAVA,PYTH,PYTH 3.6
Question

Mayur like fibonaki number.So,he gives N number to his friend Vivek and tell him to find Nth fibonaki number.

Number is very large,so find Nth fibo module by 109+7.

Input

The first line contains an integer T,total number of testcases.

For each test cases, first line N ,no of elements.
Second line N space separated numbers.

Output

Print One integer for each test cases , maximum some of chosen elements.

Constraints

1 ≤ T ≤ 1000
10 ≤ N ≤ 1018
Example

Input
2
3
5
Output
2
5
